Pizza Dude

Общий обем проведенных работы с сайтой:

  • Проработана структура сайта для удобного передвыженыя
  • разработан индивидуальный дизайн на основе фирменного стиля компании
  • адаптивная верстка позволяющася удобно пользоваться сайтом как с телефона так и с компютера
  • интеграция с CMS WordPress для возможности удобного редактирования информации на сайте через админ панель,
  • Разработка системы бонусов в виде виртуальной валюты
  • реализован функционал интернет-магазина (корзина, онлайн оплата, личный кабинет пользователя), также реализован функционал мультисайта,
  • Реализована Гугл карта с метками ближайших заведений сети пицарий Dude
  • произведена оптимизация для быстрой загрузки сайта.

О проекте

Pizza Dude - это сайт для ресторана C функцией доставки ёды. Пользователи могут выбрать блюдо и множества вариантов, оформить заказ и получить готовое блюдо с доставкой на дом. Мы создали этот сайт с нуля. дополнительно была интегрирована функция, службы поддержки.

Задание:

Разработать современный веб продукт для доставки ёды. С возможностю онлайн оплаты ,и налитичем системы бонусов

Результат:

Эффективный веб продукт

Загрузка ~1 сек, вес ~5 Мб

Редактирования ВСЕЙ информации в админ-панели
